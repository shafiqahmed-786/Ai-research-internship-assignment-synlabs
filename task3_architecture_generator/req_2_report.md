# Report for req_2

## Input requirement

Create an internal expense reporting tool for employees: submit expense claims with receipts, manager approval flow, reimbursements by finance, and monthly summary reports for accounting.

## Modules

- **Auth**: User auth and roles (employee, manager, finance)

- **Expense**: Submit expenses with receipts, edit, and view status

- **Approval**: Manager review & approve/reject flow

- **Finance**: Reimbursement processing and accounting export

- **Reporting**: Monthly summaries for accounting

### submit_expense

### approve_expense
